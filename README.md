## 策略概述

本策略是一个**全市场多资产趋势轮动**系统，每日动态选取趋势最强的标的进行全仓配置。策略在 **VnPy** 开源量化框架下开发，使用中国ETF市场的日频数据进行回测验证。

| 项目 | 说明 |
|------|------|
| 框架 | VnPy |
| 数据频率 | 日频 |
| 市场 | 中国ETF市场 |
| 调仓频率 | 每日 |
| 仓位管理 | 全仓单一标的 |

## 策略逻辑

### Step 1：风控过滤

剔除近 **3个交易日** 累计跌幅超过 **-5%** 的标的，避免短期剧烈回调资产进入候选池。

---

### Step 2：趋势周期自适应判断

使用 **ATR（Average True Range，平均真实波幅）** 衡量当前市场波动状态，动态选择拟合周期：

| 条件 | 拟合周期 |
|------|----------|
| `ATR(60) > ATR(20)` | 使用 **20日** 趋势拟合（短期） |
| `ATR(60) < ATR(20)` | 使用 **60日** 趋势拟合（长期） |

> 逻辑说明：当长期波动率高于短期波动率时，市场处于趋势启动阶段，采用短期参数捕捉早期趋势；反之采用长期参数过滤噪声。

---

### Step 3：趋势打分因子

对每个标的计算综合趋势得分：

| 因子 | 含义 |
|------|------|
| `slope` | 线性回归斜率（使用Step 2确定的周期） |
| `e^(252×slope) - 1` | 年化收益率估算 |
| `R²` | 回归拟合优度，衡量趋势强度（0~1） |

> 逻辑说明：斜率反映趋势方向与幅度，R²反映趋势的确定性。两者相乘，既追求高收益潜力，也追求趋势的稳定性。

---

### Step 4：每日调仓

- 计算所有标的当日 **Score**
- 选取 **Score最高** 的标的
- **全仓** 买入该标的

## ⚠️ 风险提示

1. **过往业绩不代表未来表现**，回测结果存在过拟合风险
2. 策略为**全仓单一标的**，集中度风险较高
3. 最大回撤约 **-34.52%**（2020年），需有较强风险承受能力
4. 策略在**震荡市中可能频繁调仓**，产生较高交易成本
5. 回测未考虑**滑点、手续费、停牌**等现实因素
